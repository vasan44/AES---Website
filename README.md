# AES Project

The project is split into separate folders:

```text
frontend/   React + Vite website and admin UI
backend/    Express API server
database/   MongoDB setup notes and env example
```

## Frontend

```bash
cd frontend
npm run dev
```

Default URL: `http://localhost:5173`

## Backend

```bash
cd backend
npm run dev
```

Default URL: `http://localhost:5000`

## Database

MongoDB configuration is documented in `database/README.md`.
